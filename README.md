# City-wise-Sales-Target-Achievement-Dashboard
An interactive Excel-based sales analytics dashboard to track salesperson performance and target achievement across major Indian cities.

Features:
City-wise sales filtering using slicers.

Salesperson-level total sales analysis.

Target Hit % tracking to measure performance.

Dynamic pivot tables and charts (bar, pie, line).

Easy-to-use UI for business stakeholders.

This dashboard supports quick performance comparison and data-driven sales strategy planning.

Purpose:
The purpose of this dashboard is to provide a clear view of sales performance and target achievement by city and salesperson. It enables managers to monitor KPIs, compare regions, identify high and low performers, and take corrective actions using interactive filters and visuals.
